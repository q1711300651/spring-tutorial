Настройка автозаполнения

Автозополнение повзоялет спрингу автоматически подбирать нужную зависимость к свойству бина.

Параметры автозаполенеия:

    no - (По умолчанию) необходимо вручную указывать зависимости.
    byName - Контейнер ищет бин с названием как названия свойства которому нужно внедрить зависимость.
    byType - подставит занчение свойства по типу с бином. Если бинов данного типа будет несколько будет брошено
             исключение, если не одно то буден присвоен null
    constructor - аннологично по типу, только для аргументов конструктора


Ограничение и недостатки

    1. Нельзя применить автозаполнение к простым свойствам типа String и т.п
    2. Не явное связывание
    3. Информация по автозаполнение, может быть не доступна для утил, формирующих документацию
