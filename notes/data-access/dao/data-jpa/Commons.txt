Поддержка нескольких репозиторев (бинов с аннотацией @Repository )

xml:
        <jpa:repositories base-package="com.acme.repositories" />

entity-manager-factory-ref
    Если использываеться несколько разных менеджеров сущностей, для каждой группы репозиториев.
    Если не указан, выполняеться поиск по апп контексту бина с именем "entityManagerFactory"
transaction-manager-ref
    Указать какой менеджер транзакции будет использовавн для работы с групой репоизториев.
    Обычно стоит указывать, когда несколько менеджеров транзакции определенно в приложении
    по умполчанию устранавливает связь с единственным определенным PlatformTransactionManager в
    контексте.
